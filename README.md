# 🚀 Simple API Project

A custom-built Python API demonstrating backend development, quality engineering practices, and automated testing.

---

## 📌 Overview

This project was designed to simulate a real-world API system with a focus on:

- Data validation
- API reliability
- Testability
- Automation readiness
- Persistent data storage

It reflects how I approach quality as a **QA Engineer — not just testing, but building systems that are testable, observable, and scalable.**

---

## ⚙️ Features

✅ Full CRUD API  
✅ Input validation and error handling  
✅ API key authentication  
✅ Persistent storage using JSON  
✅ Filtering and search capabilities  
✅ Logging for debugging and traceability  
✅ Automated API testing with Pytest  

---

## 🧰 Tech Stack

- Python (standard library)
- HTTPServer (custom API implementation)
- Pytest (automated testing)
- curl (manual/API testing)
